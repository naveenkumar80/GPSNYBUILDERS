## 📧 Contact Form Setup

To enable the contact form:

1. **Option 1: Formspree (Easy)**
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```

2. **Option 2: Netlify Forms**
   Add `netlify` attribute to form tag:
   ```html
   <form name="contact" netlify>
   ```

3. **Option 3: Custom Backend**
   Update `js/form-validation.js` with your API endpoint

## 🖼️ Image Optimization

Images should be:
- Format: WebP (with JPEG fallback)
- Max width: 1920px for banners, 800px for thumbnails
- Compressed using TinyPNG or ImageOptim
- Descriptive filenames (e.g., `concrete-foundation-project.jpg`)
